Product Import Template

Use this template to bulk upload products to the system. Please follow the instructions below:

  • Download the template Excel file using the button below.
  • Fill in the product details in the Excel file. Do not modify the header row.
  • Required fields: title, category, cost_price, selling_price
  • Category should be the exact category name (e.g., "Electronics", "Home Appliances").
  • Save the file and upload it using the bulk upload form in the application.

About Product Codes

Product codes are automatically generated based on the category name using the following logic:

  1. Take the category name and remove all vowels
  2. Keep the first 3 consonants (or pad with the first letter if fewer than 3 consonants)
  3. Append a random 3-digit number

For example, if the category is "Electronics", the product code might be "LCT-123".
